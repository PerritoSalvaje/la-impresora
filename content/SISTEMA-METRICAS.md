## PARTE 4: AUTOMATIZACIÓN DE MÉTRICAS

### 4.1 QUÉ PUEDE EXPORTAR BEEHIIV AUTOMÁTICAMENTE

Beehiiv tiene API REST y exportaciones nativas. Estas métricas se pueden extraer sin trabajo manual:

**Exportación manual desde el dashboard (semanal, 2 minutos):**
- Suscriptores totales y nuevos — Panel principal
- Open rate y click rate por email — Sección "Emails"
- Bajas — Sección "Subscribers" filtrando por "unsubscribed"
- Referidos del programa — Sección "Referral Program"

**Via API de Beehiiv (requiere token):**
- Endpoint `GET /publications/{id}/subscriptions` — lista completa de suscriptores con fecha y fuente
- Endpoint `GET /publications/{id}/posts` — métricas por email (open rate, clicks, etc.)
- Documentación: `https://developers.beehiiv.com`
- Se puede conectar a Google Sheets via Apps Script o a Make/Zapier

**Automatización recomendada con Google Apps Script:**
```javascript
// Ejecutar este script cada lunes a las 9 AM via trigger de Google Apps Script
function fetchBeehiivMetrics() {
  const API_KEY = 'tu_api_key_de_beehiiv';
  const PUBLICATION_ID = 'tu_publication_id';
  
  const url = `https://api.beehiiv.com/v2/publications/${PUBLICATION_ID}/subscriptions?status=active`;
  
  const options = {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    }
  };
  
  const response = UrlFetchApp.fetch(url, options);
  const data = JSON.parse(response.getContentText());
  
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Beehiiv');
  const row = [new Date(), data.total_results]; // suscriptores activos
  sheet.appendRow(row);
}
```

```javascript
function fetchYouTubeMetrics() {
  const channelId = 'UCxxxxxxxxxxxxxxx'; // ID del canal
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('YouTube');
  
  // Suscriptores
  const channelResponse = YouTube.Channels.list('statistics', {
    id: channelId
  });
  
  const stats = channelResponse.items[0].statistics;
  
  const row = [
    new Date(),
    stats.subscriberCount,
    stats.viewCount,
    stats.videoCount
  ];
  
  sheet.appendRow(row);
}
```

### 4.4 FÓRMULAS DE SEMÁFORO EN GOOGLE SHEETS

Usar formato condicional en la pestaña ALERTAS para colorear automáticamente:

**Para el open rate (ejemplo):**
- Valor en celda B2
- Regla 1: Si B2 < 30 → rojo (#FF4444)
- Regla 2: Si B2 >= 30 y < 40 → amarillo (#FFBB33)
- Regla 3: Si B2 >= 40 → verde (#00C851)

**Fórmula para columna "Estado" (texto):**
```
=IF(B2<30,"🔴 ALARMA",IF(B2<40,"🟡 ATENCIÓN","🟢 OK"))
```

**Tabla de umbrales para copiar:**

| KPI | Celda ref | Amarillo | Rojo | Fórmula |

### 4.5 ALERTAS AUTOMÁTICAS VÍA GOOGLE APPS SCRIPT

Este script revisa los KPIs cada lunes y te manda un email si algo está en rojo:

```javascript
function verificarAlertas() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('ALERTAS');
  const emailDestino = 'ignacio@allintokengroup.com';
  
  const alertasRojas = [];
  const alertasAmarillas = [];
  
  // Leer la columna E (estado) desde fila 2 hasta la última con datos
  const ultimaFila = sheet.getLastRow();
  
  for (let i = 2; i <= ultimaFila; i++) {
    const kpi = sheet.getRange(i, 1).getValue();
    const valor = sheet.getRange(i, 2).getValue();
    const estado = sheet.getRange(i, 5).getValue();
    const accion = sheet.getRange(i, 6).getValue();
    
    if (estado === 'ROJO') {
      alertasRojas.push(`• ${kpi}: ${valor} → ${accion}`);
    } else if (estado === 'AMARILLO') {
      alertasAmarillas.push(`• ${kpi}: ${valor}`);
    }
  }
  
  // Solo enviar email si hay algo en amarillo o rojo
  if (alertasRojas.length > 0 || alertasAmarillas.length > 0) {
    let mensaje = 'Resumen de alertas de La Impresora:\n\n';
    
    if (alertasRojas.length > 0) {
      mensaje += '🔴 EN ALARMA (acción inmediata):\n';
      mensaje += alertasRojas.join('\n') + '\n\n';
    }
    
    if (alertasAmarillas.length > 0) {
      mensaje += '🟡 EN ATENCIÓN (monitorear):\n';
      mensaje += alertasAmarillas.join('\n') + '\n\n';
    }
    
    mensaje += 'Ver dashboard completo: [URL del Google Sheet]';
    
    MailApp.sendEmail({
      to: emailDestino,
      subject: `La Impresora — Alertas KPI semana ${new Date().toLocaleDateString('es-AR')}`,
      body: mensaje
    });
  }
}

// Para configurar el trigger automático (ejecutar una sola vez):
function configurarTriggerSemanal() {
  ScriptApp.newTrigger('verificarAlertas')
    .timeBased()
    .onWeekDay(ScriptApp.WeekDay.MONDAY)
    .atHour(9)
    .create();
}
```
